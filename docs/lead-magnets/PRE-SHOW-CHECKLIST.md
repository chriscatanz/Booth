# The Complete Pre-Show Checklist

*Everything you need to do before, during, and after a trade show.*

---

## 12 Weeks Out

### Registration & Planning
- [ ] Register for booth space
- [ ] Confirm booth size and location
- [ ] Review exhibitor manual/kit
- [ ] Note all deadlines in calendar
- [ ] Book hotel rooms (before block fills)
- [ ] Assign booth staff

### Budget
- [ ] Create show budget
- [ ] Get PO/approval for major expenses
- [ ] Set up vendor accounts

---

## 8 Weeks Out

### Booth & Display
- [ ] Confirm booth design/layout
- [ ] Order graphics and signage
- [ ] Rent/reserve furniture
- [ ] Order floor/carpet if needed
- [ ] Plan lighting and A/V

### Collateral
- [ ] Order brochures/datasheets
- [ ] Order business cards
- [ ] Design/order giveaways
- [ ] Prepare demo materials

### Logistics
- [ ] Get freight quote
- [ ] Schedule pickup
- [ ] Order show services (electric, internet, etc.)

---

## 4 Weeks Out

### Shipping & Drayage
- [ ] Confirm advance warehouse address
- [ ] Note material handling deadline
- [ ] Arrange crating if needed
- [ ] Insure shipment
- [ ] Ship booth to advance warehouse (if applicable)

### Travel
- [ ] Book flights
- [ ] Confirm hotel reservations
- [ ] Arrange ground transportation
- [ ] Plan team dinner/activities

### Lead Capture
- [ ] Order lead retrieval scanners
- [ ] Test lead capture app
- [ ] Prepare qualification questions
- [ ] Set up CRM integration

---

## 2 Weeks Out

### Final Prep
- [ ] Confirm all shipments
- [ ] Get tracking numbers
- [ ] Final booth staff briefing
- [ ] Review booth schedule/shifts
- [ ] Prepare travel packets

### Packing
- [ ] Pack auxiliary items
- [ ] Create packing list
- [ ] Prepare tool kit
- [ ] Pack emergency supplies

### Communications
- [ ] Send pre-show emails to customers
- [ ] Announce on social media
- [ ] Brief sales team on talking points
- [ ] Schedule customer meetings

---

## 1 Week Out

### Confirmations
- [ ] Confirm booth ready at venue
- [ ] Verify all services ordered
- [ ] Confirm labor ordered
- [ ] Final headcount for travel

### Materials Check
- [ ] Print backup materials
- [ ] Charge all devices
- [ ] Download presentations offline
- [ ] Test demo equipment

---

## Day Before

### Logistics
- [ ] Confirm flight times
- [ ] Check in online
- [ ] Review booth setup schedule
- [ ] Locate advance warehouse on-site

### Packing
- [ ] Pack carry-on with essentials
- [ ] Bring copies of all confirmations
- [ ] Pack comfortable shoes
- [ ] Bring business cards

---

## Show Day Setup

### Before Opening
- [ ] Arrive early for setup
- [ ] Inventory all shipments
- [ ] Report any damage immediately
- [ ] Set up booth per plan
- [ ] Test electricity and internet
- [ ] Position all displays
- [ ] Stock brochures and giveaways
- [ ] Brief booth staff

### Quick Checks
- [ ] All lights working?
- [ ] Monitors displaying correctly?
- [ ] Lead scanners charged?
- [ ] Staff schedule posted?
- [ ] Emergency contact info shared?

---

## During the Show

### Daily Opening
- [ ] Restock supplies
- [ ] Check lead scanner battery
- [ ] Brief staff on day's goals
- [ ] Check for any customer meetings

### Hourly
- [ ] Keep booth tidy
- [ ] Cycle through breaks
- [ ] Capture leads properly
- [ ] Take notes on hot prospects

### Daily Close
- [ ] Secure valuable items
- [ ] Count remaining collateral
- [ ] Back up lead data
- [ ] Debrief team
- [ ] Plan for tomorrow

---

## Post-Show (On-Site)

### Teardown
- [ ] Supervise breakdown
- [ ] Inventory all items
- [ ] Note any damage
- [ ] Confirm return shipment
- [ ] Keep tracking number
- [ ] Tip labor if appropriate

### Immediate Follow-Up
- [ ] Export all leads
- [ ] Send "thanks for visiting" email
- [ ] Hand off hot leads to sales
- [ ] Quick social media post

---

## Post-Show (Back Home)

### Within 48 Hours
- [ ] Complete lead upload to CRM
- [ ] Send personalized follow-ups
- [ ] Brief sales on key opportunities
- [ ] Post on social media

### Within 1 Week
- [ ] Complete expense report
- [ ] Reconcile budget
- [ ] Verify return shipment arrived
- [ ] Inspect booth for damage
- [ ] Write show recap

### Within 1 Month
- [ ] Calculate initial ROI
- [ ] Review lead conversion
- [ ] Gather team feedback
- [ ] Document lessons learned
- [ ] Decide: Do this show again?

---

## Emergency Kit Checklist

Pack in carry-on or hand-carry to show:

- [ ] Power strip (6+ outlets)
- [ ] Extension cord (25 ft)
- [ ] Blue painter's tape
- [ ] Zip ties (various sizes)
- [ ] Scissors
- [ ] Box cutter
- [ ] Screwdriver set
- [ ] Level (small)
- [ ] Cleaning wipes
- [ ] Lint roller
- [ ] Backup chargers (phone, laptop)
- [ ] First aid kit
- [ ] Pain relievers
- [ ] Breath mints
- [ ] Snacks
- [ ] Cash (for tips)
- [ ] Copies of all confirmations
